## 2026-02-27 - Icon Ligatures and Responsive Buttons ARIA Labels
**Learning:** Material Symbols (and similar icon fonts) that use text ligatures (e.g., `<span class="material-symbols-outlined">rocket_launch</span>`) will be read aloud by screen readers as the literal ligature text (e.g., "rocket underscore launch") if not hidden. Additionally, buttons that hide their visible text on small screens via responsive classes (e.g., `hidden lg:inline`) lose their accessible name for screen readers on those smaller viewports.
**Action:** Always add `aria-hidden="true"` to structural icon elements (like `<span class="material-symbols-outlined">`) to prevent screen readers from announcing raw ligature text. Furthermore, ensure that any interactive element (`<button>` or `<a>`) where the text might be visually hidden at certain breakpoints has an explicit `aria-label` attribute to maintain a consistent accessible name across all viewport sizes.
